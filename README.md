## ✨ 核心功能
- 🖱️ 右键快捷添加/移除网页链接
- ⌨️ 快捷键添加（默认 `Alt+S`，添加到默认分类）
- 🧩 简洁的管理界面
- 🗂️ 多分类管理（支持自定义分类）
- 🔍 全文搜索（标题/URL）



## 🚀 安装使用
```bash
# 安装依赖
npm install

# 开发模式
npm run start

# 构建生产版本
npm run build
```

1. 打开 Chrome 浏览器 → 进入 `chrome://extensions/`
2. 启用「开发者模式」
3. 点击「加载已解压的扩展程序」→ 选择项目 `build` 目录
